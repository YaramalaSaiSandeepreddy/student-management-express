# Student CRUD App (Node.js + Express)

A simple Student Management web application built using Node.js and Express.  
This app allows users to add, update, and delete student records stored in a JSON file.

## Features

- Add new students
- Update student course
- Delete students
- Simple HTML interface
- Data stored locally in JSON file
- Lightweight Express server

## Tech Stack

- Node.js
- Express.js
- HTML
- JSON file storage
